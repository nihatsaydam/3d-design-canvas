# Menu Project Handoff

Bu kopya sadece ana menu/site tarafi icindir. Editor kaynak kodlari bu projeden cikarildi.

Calisilacak alanlar:

- `src/pages/Index.tsx`
- `src/pages/Gallery.tsx`
- `src/pages/Blog.tsx`
- `src/pages/Pricing.tsx`
- `src/pages/Login.tsx`
- `src/pages/Signup.tsx`
- `src/components/site/`
- `src/components/auth/`
- `src/assets/`

Mumkunse dokunulmamasi gereken ortak alanlar:

- `src/App.tsx`
- `src/main.tsx`
- `src/components/ui/`
- `src/hooks/`
- `src/lib/`
- `src/index.css`
- `package.json`

`/edit` adresi bu kopyada sadece placeholder sayfaya gider. Final projede asil editor moduluyle birlestirilmelidir.

Final projeye geri alirken oncelikli olarak su dosya/klasorler kopyalanir:

- `src/pages/Index.tsx`
- `src/pages/Gallery.tsx`
- `src/pages/Blog.tsx`
- `src/pages/Pricing.tsx`
- `src/pages/Login.tsx`
- `src/pages/Signup.tsx`
- `src/components/site/`
- `src/components/auth/`
- `src/assets/`

Calistirma:

```bash
npm install
npm run dev
```

Bu kopya varsayilan olarak `http://localhost:8082` adresinde acilir.
